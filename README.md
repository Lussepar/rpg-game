# Text RPG (C# Console Game)

This is a simple text-based RPG made in C# as a beginner learning project.

## About the Game

You start in a dungeon with no memory of how you got there.  
You create a character, assign stats, and your choices affect how the story plays out.

## Features

- Character creation with skill points
- Simple class system based on stats
- Story-driven dungeon exploration
- Choice-based gameplay
- Console-based interface

## How to Play

1. Run the program in Visual Studio (or any C# IDE)
2. Choose "Play" from the main menu
3. Create your character and assign stats
4. Make choices to progress through the story

## Controls

- Type numbers for menu choices
- Type letters like `A`, `H`, `D`, `Y`, `N`, `I`, `E`, `S` when prompted
- Press Enter after each input

## Classes

Your class depends on your stats:

- High Attack → Berserker  
- High Defence → Paladin  
- High Health → Juggernaut  
- Balanced stats → Adventurer
- (More classes to come)
